# Documentation

The current source of truth for this repository lives in:

- `CLAUDE.md` for architecture, workflows, and agent guidance
- `docs/HOWTO_SYSTEMD.md` for the production systemd deployment used on this host

Historical implementation notes, experiments, bugfix writeups, AI assistant memos, and planning documents have been moved to `docs/history/`.

Treat `docs/history/` as repo archaeology:

- It is useful for context and past decisions
- It is not guaranteed to reflect the current codebase
- Old paths, status notes, and assumptions may be outdated
